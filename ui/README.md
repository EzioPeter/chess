# 中国象棋 App 界面

这是一个零依赖的静态前端页面，现在已经支持本地双人中国象棋对弈。

## 使用方式

直接在浏览器打开 `index.html` 即可。

如果你想用本地服务打开，也可以在当前目录执行：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 当前包含

- 中国象棋标准初始摆盘
- 本地双人真走棋
- 合法走法校验
- 选子高亮与可落点提示
- 吃子、将军、将死、将帅照面规则处理
- 悔棋、提示着法、重新开局
- 木纹 / 玉石双主题切换
- 棋子信息面板
- 桌面端和移动端自适应布局
