# 中国象棋 App 界面

这是一个零依赖的中国象棋前端页面，现在已经支持：

- 本地双人对弈
- 连接 `../Pikafish` 引擎进行人机对战

## 使用方式

如果你只想体验本地双人模式，直接在浏览器打开 `index.html` 即可。

如果你想启用 `Pikafish` 人机对战，请先确认引擎已经编译完成：

```bash
cd ../Pikafish/src
make -j4 build
```

然后在 `ui` 目录启动 bridge：

```bash
node server.js
```

或者：

```bash
npm start
```

默认访问地址是 `http://127.0.0.1:3000`。

## 当前包含

- 中国象棋标准初始摆盘
- 本地双人真走棋
- Pikafish 人机对战
- 合法走法校验
- 选子高亮与可落点提示
- 吃子、将军、将死、将帅照面规则处理
- 悔棋、提示着法、重新开局
- 引擎提示着法
- 引擎执红 / 执黑切换
- 引擎思考时间切换
- 木纹 / 玉石双主题切换
- 棋子信息面板
- 桌面端和移动端自适应布局
